# MedSense

MedSense is a React + Vite frontend and Express + MySQL backend.

- Frontend: `frontend/`
- Backend: `backend/`

Production target:

- Frontend: `https://www.mymedsense.co`
- API: `https://api.mymedsense.co`

## Deploy

Frontend is configured for Vercel in `frontend/vercel.json`.
Backend is configured for Render in `backend/render.yaml`.
